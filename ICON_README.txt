📱 학원가자 아이콘 생성 방법

icon.svg 파일을 PNG로 변환해야 합니다.

=== 방법 1: 온라인 변환 (추천) ===

1. https://cloudconvert.com/svg-to-png 접속
2. icon.svg 업로드
3. 크기 설정:
   - 192x192 → icon-192.png
   - 512x512 → icon-512.png
4. 다운로드 후 프로젝트 폴더에 저장

=== 방법 2: Figma/Photoshop ===

1. SVG 파일 열기
2. 내보내기:
   - 192x192 PNG
   - 512x512 PNG
3. icon-192.png, icon-512.png로 저장

=== 임시 사용 (테스트용) ===

아이콘 없이도 PWA는 작동합니다!
배포 후 아이콘 추가 가능합니다.

Netlify 배포 시:
1. 일단 아이콘 없이 배포
2. 나중에 PNG 생성 후 추가
3. Git push하면 자동 업데이트

=== 빠른 시작 ===

아이콘 걱정하지 마세요!
바로 배포하고 나중에 추가해도 됩니다! 🚀
